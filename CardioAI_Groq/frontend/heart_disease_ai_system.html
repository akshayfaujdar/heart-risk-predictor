About the Project

Technical overview, architecture, and implementation details.

System Architecture
1
Data Preprocessing
70,000 patient records loaded, features normalized, balanced dataset ensured for unbiased training.
2
ML Model Training
4 classifiers trained: Logistic Regression, Random Forest, Decision Tree, Gradient Boosting. Best model selected automatically.
3
Risk Inference Engine
Gradient Boosting model serves real-time predictions with feature-weighted probability scores.
4
AI Chatbot Layer
Claude AI integration provides medically-informed conversational responses via the Anthropic API.
5
Frontend Dashboard
Single-page React-style application with live charts, interactive predictor, and patient history tracking.
Dataset Information

Source: EarlyMed Heart Disease Risk Dataset

Records: 70,000 patients (balanced 50/50)

Features: 18 clinical attributes (symptoms + risk factors)

Target: Heart_Risk (binary: 0 = low, 1 = high)

Missing Values: None detected

Age Range: 20–84 years (mean: 54.5)

Class Balance: Perfect 50/50 split
